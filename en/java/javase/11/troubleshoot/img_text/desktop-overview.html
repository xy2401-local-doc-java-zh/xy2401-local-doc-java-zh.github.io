Description of the illustration desktop-overview.gif

This diagram shows an overview of the Java SE desktop technologies within the context of the Java platform. It consists of a number of colored rectangular boxes.

The top element of the diagram is a green box labeled User Application.

Below it, in the middle of the diagram, is a grey box labeled Desktop Technologies which contains five smaller yellow boxes. The leftmost of the yellow boxes is labeled Deployment and the rightmost is labeled Media APIs. Between them, a box labeled Swing is placed above two smaller yellow boxes labeled AWT and Java 2D.

The bottom element of the diagram is a blue box labeled Core Tools and Libraries.
