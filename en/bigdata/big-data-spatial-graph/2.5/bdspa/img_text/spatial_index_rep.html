Description of the illustration spatial_index_rep

User data and its associated spatial index representation, as explained in text before the figure. For example, in the User Data, Split 1 consists of records r1, r2, and r3; and Split 1 has a Key-Value pair where the Key is Split 1 and the Value is the index name Rtree(r1, r2, r3).
