Oracle Big Data Connectors

Oracle Big Data Connectors is a software suite that integrates processing in Apache Hadoop distributions with operations in Oracle Database. It enables the use of Hadoop to process and analyze massive volumes of data and to use it with database data to derive new and critical business insights.
The tools in this suite include Oracle SQL Connector for Hadoop Distributed File System, Oracle Loader for Hadoop, Oracle Data Integrator, Oracle XQuery for Hadoop, Oracle R Advanced Analytics for Hadoop, and Oracle Datasource for Apache Hadoop.

Release 4.12

The current release of Big Data Connectors is 4.12. Documentation for this release is included in Oracle Big Data Appliance Online Documentation Library Release 4.12.
