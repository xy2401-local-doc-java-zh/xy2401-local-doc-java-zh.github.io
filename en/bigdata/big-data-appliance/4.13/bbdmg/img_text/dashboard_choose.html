Description of dashboard_choose.png

This image shows the Select Widget dialog box on top of the Oracle Big Data Manager dashboard. The dialog box has these tabs:

  • All

  • Jobs

  • Monitoring

  • Storages

  • Notebook

The All tab is selected, and all the widget types are listed:

  • Job History

  • Recent jobs

  • Cluster memory

  • Cluster CPU

  • Cluster notes

  • Yarn Queues

  • Storage details

  • Node widget

  • Paragraph widget

Following the list of widget types are two buttons: Add and Cancel.
