Description of data_explorer.png

This image shows the Oracle Big Data Manager Data Explorer page.

At the top of the page is a tab bar with these tabs:

  • Home

  • Data

  • Jobs

  • Monitoring

  • Notebook

  • Dashboard

On the left side of the page are these tabs:

  • Explorer

  • Storages

The Data tab on the top of the page is selected, and the Explorer tab on the left is selected.

The page contains two panels that display hierarchies of files and directories.

The first panel contains:

  • A Storage drop down list with HDFS Storage (hdfs) selected.

  • A toolbar with the following icons:

    • Refresh

    • Icons

    • Copy

    • Copy her from HTTP(S)

    • Move

    • Delete

    • Compare

    • Hidden files are not displayed

    • New directory

    • File upload

  • A list of files and directories.

The second panel contains:

  • A Storage drop down list with Oracle Object Storage Classic (bdcs) selected.

  • A toolbar with the following icons:

    • Refresh

    • Icons

    • Copy

    • Copy her from HTTP(S)

    • Compare

    • Hidden files are not displayed

    • New directory

  • A list of files and directories.
