Oracle Big Data Appliance

Oracle Big Data Appliance is a high-performance, secure platform for running diverse workloads on Hadoop and NoSQL systems. When coupled with Oracle Big Data SQL, Oracle Big Data Appliance extends Oracle SQL to Hadoop and NoSQL systems. Oracle Big Data Appliance is pre-configured for secure environments leveraging Apache Sentry, Kerberos, and both network encryption and encryption of data at rest.
