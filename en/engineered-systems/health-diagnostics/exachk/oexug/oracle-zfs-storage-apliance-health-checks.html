2.5 Running Oracle ZFS Storage Appliance Health Checks

Oracle ORAchk supports health checks for Oracle ZFS Storage Appliances.

To run Oracle ORAchk on one Oracle ZFS appliance, use the -zfssa option.

To run Oracle ORAchk on multiple Oracle ZFS appliances, specify a comma-delimited list of Oracle ZFS Storage Appliances:
./orachk -zfssa node1,node2
