Preface

This guide explains how to use Oracle ORAchk and Oracle Exachk diagnostic tools on Oracle Database Engineered Systems.

It also explains the prerequisites to install and configure Oracle ORAchk and Oracle EXAchk diagnostic tools.

Audience

Oracle® Database ORAchk and EXAchk User's Guide provides conceptual and usage information about the diagnostic tools for the database administrators.

This guide assumes that you are familiar with Oracle Database concepts.
