3.1 Scope and Supported Platforms for Running Oracle EXAchk on Oracle Engineered Systems

Oracle EXAchk supports all supported hardware types, operating systems, firmware versions, and Oracle versions of the following Oracle engineered systems.

  • Oracle Big Data Appliance

  • Oracle Exadata Database Machine Version 2 and later

  • Oracle Exalogic

  • Oracle Exalytics

  • Oracle SuperCluster

  • Zero Data Loss Recovery Appliance

  • Oracle Private Cloud Appliance
