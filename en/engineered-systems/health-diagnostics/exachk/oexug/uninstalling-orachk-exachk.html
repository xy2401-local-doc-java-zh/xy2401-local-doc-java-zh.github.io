1.13 Uninstalling Oracle ORAchk and Oracle EXAchk

Note:

Oracle ORAchk daemon functionality is supported only on the Linux and Solaris operating systems.
  1. Stop any active Oracle ORAchk or Oracle EXAchk daemon clients:
    orachk -d stop_client
exachk -d stop_client
  2. Stop any running Oracle ORAchk or Oracle EXAchk daemon:
    orachk -d stop
exachk -d stop
  3. Remove the Oracle ORAchk or Oracle EXAchk daemon from automatic restart configuration:
    orachk -initrmsetup
exachk -initrmsetup
  4. Delete the Oracle ORAchk or Oracle EXAchk install directory.
