This image depicts a flow of events that occur for automatic collections.

A significant problem occurs in the Oracle Grid Infrastructure or Oracle Databases.

Oracle Trace File Analyzer automatically detects the event, and then collects and packages relevant diagnostics. It then notifies the relevant DBA or Sys Admin by email.

The DBAs or Sys Admins can then upload the collection to Oracle Support for further help.
