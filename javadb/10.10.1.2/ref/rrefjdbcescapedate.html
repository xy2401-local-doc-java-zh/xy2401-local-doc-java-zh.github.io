JDBC escape syntax for date formats

Derby interprets the JDBC escape syntax for dates as equivalent to the correct SQL syntax for dates.

Syntax

{ d 'yyyy-mm-dd' }

Equivalent to

DATE('yyyy-mm-dd')

Example

VALUES {d '2010-10-19'}
